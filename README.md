# LexiSprint AI 背单词 App

一个零依赖的单页背单词应用，支持：

- 手动维护词库
- 按“认识 / 模糊 / 不认识”进行每日复习
- 基于熟悉度安排后续复习
- 统计今日学习词与已掌握词
- 根据“今日新词 + 已掌握基础词 + 目标考试”生成长句或短文
- 支持离线演示模式和 OpenAI 兼容 API 模式

## 启动方式

### 方式一：直接打开

直接用浏览器打开 `index.html` 即可使用。

### 方式二：本地静态服务

```bash
python server.py
```

然后访问 <http://127.0.0.1:8000>

## 使用流程

1. 添加单词，或点击“载入示例词库”。
2. 在“今日背词”里完成一轮学习。
3. 选择考试类型，比如 CET-4、IELTS、考研英语。
4. 在“长句 / 文章生成”里输入主题并点击生成。
5. 如果想接真实大模型，把生成模式切到 `API 模式`，填写：
   - `API Base URL`
   - `API Key`
   - `模型名称`

## AI 生成逻辑

- 离线演示模式：不联网，使用当前学习记录拼装一份可演示的考试风格内容。
- API 模式：调用 OpenAI 兼容的 `/chat/completions` 接口生成真正动态的内容。
